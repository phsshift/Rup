# Сборка APK через GitHub Actions

1. Загрузить все файлы проекта в корень репозитория.
2. Проверить, что есть файл `.github/workflows/main.yml`.
3. Открыть вкладку `Actions`.
4. Выбрать `Build Android APK`.
5. Нажать `Run workflow`.
6. После зелёной галочки открыть запуск и скачать APK в блоке `Artifacts`.

Если сборка падает, открой job `build` и скопируй последние 30–50 строк лога.
